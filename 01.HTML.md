# HTML

#### 입력 양식 태그

##### form 태그

- method: http 메서드
  - get(default): url로 값 전달(URL Encoding), 길이 제한
  - post: 비공개(request body)로 값을 전달, 제한x
- action: 전송할 url
- input
  - name,value: query string에서 쓰임

##### 확인방법

개발자 도구 network탭 > 요청한 request와 각 헤더 정보, payload에서 전송된 쿼리스트링을 확인할 수 있음

#### 문서 구조화 (semantic tag)

- 블럭형 태그(div) / 인라인 태그(span)
- 시멘틱 웹이 의미를 인식하는 태그
  - 검색엔진이 페이지를 분석할때 가중치에 차이가 있음

##### 검색엔진 최적화

검색엔진?
:웹에서 원하는 정보를 찾는데 걸리는 시간을 최소화하기 위해 사용되는 시스템

bot, 크롤러
: 사용자가 검색했을때 원하는 결과를 빠르게 보여주기 위해 정보를 미리 데이터베이스화 시키는 프로그램

검색엔진 최적화(SEO)란?
: 검색이 잘 되도록 설계하는 것
html로 SEO 하는 방법 2가지

1. 메타 태그: 검색엔진에 정보를 제공할 목적인 태그
2. 시맨틱 태그: 웹페이지 구조를 구성하는 태그
   - header, nav, section, article, aside, footer//'
