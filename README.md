### 커밋 메세지 컨벤션

<code>feat</code> : 새로운 기능 추가  
<code>fix</code> : 버그 수정  
<code>docs</code> : 문서 수정  
<code>style</code> : 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우  
<code>refactor</code> : 코드 리펙토링  
<code>test</code> : 테스트 코드, 리펙토링 테스트 코드 추가  
<code>chore</code> : 빌드 업무 수정, 패키지 매니저 수정
