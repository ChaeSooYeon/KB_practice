# Vue.Js

## 특징

- SPA
- data driven형식: data가 바뀔때 뭔가를 하겠다
- Javascript 기본 형식에 제일 가까운 프레임워크
- MVVM 패턴
  - Model - View - ViewModel
- 가상 DOM
  - 브라우저가 가상DOM을 활용해 변경된 부분을 탐지하여 data가 바뀌면 그 부분만 DOM을 새로 렌더링 함 ==> 빠른 UI렌더링 성능을 제공
  - ==> '=' 연산자가 실행될 때

## 프로젝트 구조

### 파일 구조 설명 -index.html

```html
<!doctype html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    <div id="app">
      <h2>{{message}}</h2>
    </div>
    <!-- vue라이브러리 호출 -->
    <script src="https://unpkg.com/vue"></script>
    <script>
      let model = { message: 'Hello Vue3!' };
      // view model 생성
      let vm = Vue.createApp({
        // 옵션 객체
        name: 'App',
        data() {
          return model;
        },
      }).mount('#app'); /// mount: 'app'과 view model을 연결하여 렌더링 하겠다. viwe model의 관리대상으로 설정하는 것
    </script>
  </body>
</html>
```

- `id="app"` : Vue의 관리 대상으로 설정
