## 변수선언: let 과 const

- const: 상수로 선언.
  - object를 상수로 선언 시?
  - 참조값이 상수이고, 내부값의 변경은 가능함.
- let: 참조값을 변경 가능

## 구조분해 할당

```js
// ex1)
let arr = [10, 20, 30, 40];
let [a1, a2, a3] = arr; // 순서에 따라 맵핑
console.log(a1, a2, a3); // 10, 20, 30
```

```js
//ex2)
let p1 = { name: 'kim', age: 20, gender: 'M' };
let { name: n, age: a, gender } = p1; //이름에 따라 맵핑, 별칭 지정 가능
console.log(n, a, gender); // kim, 20, M
```

## 함수 선언

- parameter: 함수 내부에서 사용하는 변수로 전달받는 인자
- argument: 함수내부로 전달하는 인자. 전달 값
- return value: 함수에 정의된 기능을 수행 후 return 되는 값

```js
// 1. 함수선언문으로 정의하여 호출
function foo() {
  console.log('foo');
}
let foo2 = foo; //식별자
foo();
foo2(); // 'foo2'라는 식별자로 호출 ---> call by Reference
// 2. 함수 리터럴 표현식으로 함수 호출 - IIFE구조 (매크로에서 활용)
(function bar() {
  console.log('bar');
})();
// 3. 화살표 함수
const add = (x, y) => {
  console.log(x + y);
};
add(1, 2);
```

- 함수에서 객체는 참조에 의한 전달(call by Reference) 방식으로 동작한다

## 중첩 함수 == 내부 함수

- 함수 내부에 정의된 함수(nested function, inner function)
- 외부함수(outer function) 내부에서만 호출할 수 있음
- 자신을 포함한 외부함수를 돕는 helper의 역할을 함

```js
function outer() {
  let x = 1;

  function inner() {
    let y = 2;
    console.log(x + y);
  }
}
inner(); //3
```

## 고차함수(Higier-Order Function, HOF)

- 정의: 매개변수를 통해 다른 함수의 내부로 전달되는 함수
- 콜백함수는 고차 함수에 의해 호출된다.

```js
function repeat(n,f){
  for(let i=0; i<n; i++>){
    logOdd(i);
  }
}
function logAll(i){
  console.log(i)
}
function logOdd(i){
  return if(i%2) console.log(i);
}
repeat(5, logAll)// 0,1,2,3,4
repeat(5,logOdd);// 1,3
```
